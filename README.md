# AdventureWorks-Analytics-Dashboard
A comprehensive Excel data analysis project using the AdventureWorks dataset. Features automated data cleaning, pivot tables, KPI tracking, and an interactive dashboard built with slicers and charts. Designed to showcase sales performance insights, business trends, and advanced Excel skills.

## Overview
This project demonstrates a complete data analysis and visualization workflow using Microsoft Excel.  
It uses the AdventureWorks dataset, a popular sample database that represents a manufacturing and retail business.  
The goal is to transform raw sales data into actionable insights through an interactive dashboard that supports data-driven decision making.

## Features
- **Data Cleaning and Preparation:** Cleaned and standardized raw data using Power Query and Excel formulas.  
- **Exploratory Data Analysis:** Used pivot tables and charts to uncover sales trends by product, region, and time.  
- **Interactive Dashboard:** Designed an automated dashboard featuring slicers, charts, and KPI summaries such as total revenue, profit margin, and year-over-year growth.  
- **Automation and Efficiency:** Incorporated macros and dynamic formulas to refresh and update insights automatically.  
- **Visual Storytelling:** Built a clear layout for monitoring business performance at a glance.

## Tools and Skills Demonstrated
- Microsoft Excel (Pivot Tables, Power Query, Power Pivot, VBA)
- Data Cleaning and Transformation
- KPI Design and Performance Tracking
- Dashboard and Report Development
- Business Intelligence and Analytical Thinking

## Key Insights
- Highlighted top-performing product categories and regions.
- Revealed seasonal sales patterns and growth opportunities.
- Identified profit-driving and underperforming areas within the business.

## How to Use
1. Clone or download the repository.  
2. Open the Excel file `AdventureWorks Sales Dashboard.xlsm`.  
3. Enable macros when prompted.  
4. Navigate through the dashboard using slicers and filters to view performance insights.
